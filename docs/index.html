核心功能

🖥️

服务器管理

  • • 多台服务器 SSH 连接管理
  • • 支持跳板机/堡垒机
  • • SOCKS5/SOCKS4 代理连接
  • • 服务器分组、在线状态监控
💻

Web 终端

  • • 基于 WebSocket 的实时终端
  • • 多标签页管理
  • • 鼠标选中自动复制
  • • 终端会话历史记录与回放
🪟

Desktop 客户端

  • • 基于 Wails 的 Windows 桌面客户端
  • • 内置后端服务
  • • 本地终端支持
  • • 系统托盘运行
🐳

容器管理

  • • Docker 容器查看、启动、停止
  • • 容器日志查看
  • • 镜像列表管理
🚀

服务部署

  • • Docker Compose 一键部署
  • • 自定义 compose 配置
  • • 部署状态跟踪
⏰

计划任务

  • • Crontab 可视化编辑
  • • 任务执行记录查看
  • • 一键执行任务
📁

文件管理

  • • 目录浏览、文件预览
  • • 文件上传、下载
  • • 在线编辑
  • • 拖拽上传支持
📝

笔记管理

  • • 笔记分类、搜索
  • • Markdown 格式支持
  • • 终端页面快捷查看
🔗

端口隧道

  • • SOCKS5 代理隧道
  • • 支持绑定 127.0.0.1 或 0.0.0.0
  • • 映射本地端口到远程服务器

技术栈

🐹 Go
⚡ Gin
🗄️ SQLite
🖥️ xterm.js
🎨 Alpine.js
💨 TailwindCSS
🪟 Wails

快速开始

# 编译

go build -o deploy-manager.exe ./cmd/server/main.go


# 运行

./deploy-manager.exe --password yourpass


# 访问

浏览器打开 http://localhost:3000
