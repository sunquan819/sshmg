🖥️ Desktop 专属

本地终端
像 IDE 一样操作服务器

在桌面客户端内直接打开本机 PowerShell、CMD、Git Bash，支持多标签页和分屏操作，让运维效率翻倍。

01

多标签 + 分屏

最多 4 个横向/纵向分屏，同时操作多台服务器

02

Windows Terminal 风格

鼠标选中自动复制，右键粘贴，支持滚轮滚动历史

03

AI Agent 集成

自动探测 opencode、codex 等 AI CLI，一键启动，支持历史 session 继续

04

文本跨终端发送

选中文本可发送到当前终端、其他分屏或所有终端
